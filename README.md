# To play the game:
Run the file 'main.py'

# Controls:
  Player 1 Controls:
  WASD = Movement
  E = Primary Attack
  R = Heavy Attack
  T = Misc Attack
  F = Blocking

  Player 2 Controls:
  Arrows = Movement
  , = Primary Attack
  . = Heavy Attack
  / = Misc Attack
  RShift = Blocking

# Special Notes:
  The Stick can transform into The Stick Avatar when it has below 30% health available. This transformation will boosts its stats and health, along with obtaining a new set of         moves.
